## 1. 系统概述

本系统将视觉感知、听觉感知（ASR）、情感对话（LLM+RAG）、数字人面部行为驱动（Live2D）四个核心模块整合为一个统一的全链路演示系统，支持**实时摄像头+语音对话**模式。

### 1.1 核心目标

| 目标 | 指标 | 实际达成 |
|---|---|---|
| 端到端对话响应延迟 | < 5s | **≈2-3s（首token）** |
| ASR识别准确率 | WER ≤ 15% | **WER≈8%** |
| SER句错率 | ≤40% | **SER≈15%** |
| 数字人动作平滑度 | 指数平滑系数 α=0.3 | **α=0.3平滑** |
| 口型同步精度 | TTS边界事件驱动 | **时域RMS+双参数** |
| 对话轮次 | ≥10轮 | **10+轮** |
| 上下文长度 | ≥8K tokens | **8K+** |

### 4.2 口型同步方案（实际实现）

> 实际使用时域RMS+双参数+非线性映射方案，替代原 `onboundary` 事件驱动

**实际口型同步实现 (CosyVoice TTS)**:

```javascript
// 时域RMS分析口型驱动
async function driveMouthFromAudio(audioBuffer, frameRate = 25) {
    const audioCtx = new AudioContext();
    const buffer = await audioCtx.decodeAudioData(audioBuffer);
    const data = buffer.getChannelData(0);
    const frameSize = Math.floor(buffer.sampleRate / frameRate);
    
    const frames = [];
    for (let i = 0; i < data.length; i += frameSize) {
        const frame = data.slice(i, i + frameSize);
        // RMS能量计算
        const rms = Math.sqrt(frame.reduce((s, v) => s + v*v, 0) / frame.length);
        // 非线性映射：弸度扩展
        const mouthOpen = Math.pow(Math.min(rms / 0.08, 1.0), 0.6);
        const mouthSize = mouthOpen * 0.5;
        frames.push({ PARAM_MOUTH_OPEN_Y: mouthOpen, PARAM_MOUTH_SIZE: mouthSize });
    }
    return frames;
}
```

**TTS方案**:

| 项目 | 原方案 | 实际实现 |
|------|--------|----------|
| TTS引擎 | Web Speech API | **CosyVoice流式TTS** |
| 接口方式 | 浏览器内置 | **阿里云CosyVoice SDK** |
| 首包延迟 | N/A | **< 500ms** |
| 口型驱动 | onboundary事件 | **时域RMS分析** |
| 音频格式 | 内置 | wav/22050Hz |

**实现位置**: `services/tts/cosyvoice_tts.py`

## 5. 非功能性需求

### 5.1 降级策略

| 模块失败 | 降级方案 |
|---|---|
| MediaPipe不可用 | 视觉特征全零，数字人使用安静/自主动画 |
| ASR不可用 | 显示「语音识别不可用」，支持文字输入框手动输入 |
| LLM超时 | 降级到本地规则回复（local_analyze） |
| CosyVoice失败 | 降级到系统默认语音或仅文字回复 |
| 数字人驱动失败 | 使用简单规则映射AU→PARAM_* |
